# enb

## Instale o Nodejs
```
https://nodejs.org/en
```

## Abra o cmd(prompt de comando) e acesse a pasta do projeto
```
Descompactar o arquivo do projeto
Ex.: Caso você tenha descompactado o diretório do projeto na pasta de downloads:
cd Downloads\enb
```

### Execute o comando
```
npm install
```
### Execute o comando
```
npm run serve
```

### Abra o navegador
```
https://localhost:8080
```
